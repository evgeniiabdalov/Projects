README
-----------------------------------------------------------------------------------------  

ContinuousViz application is a MAPF-R visualization tool.
Its purpose is to visualize solution to MAPF-R problem and
facilitate detection of possible redundancies of the solution.
It was made by Evgenii Abdalov as a part of the thesis work 

Visual Analysis of Plans for Multi-Agent Path Finding with Continuous Time (MAPF-R)

supervised by doc. RNDr. Pavel Surynek, Ph.D.

-----------------------------------------------------------------------------------------  

In folder 'exe' there are executable binary file as well as run scripts, test data and
Manual and README.

In folder 'src/impl' there is folder 'ContinuousViz application' where source codes could be find as well as
photo and video examples of application functioning.

In folder 'src/text' there are thesis.tex file and folder 'Thesis files'

In folder 'text' there is thesis.pdf file
